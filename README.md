# ChaoticGrid

ChaoticGrid is a collaborative, real-time "Social Bingo" game.

## Tech Stack
- **Frontend:** Angular 19 (Signals, Standalone Components, Material)
- **Backend:** .NET 10 (Minimal APIs, DDD, SignalR)
- **Database:** SQLite (EF Core)
- **Containerization:** Docker

## Conventions
- **C#:** PascalCase for all projects and namespaces.
- **Angular:** kebab-case for all files and selectors.
